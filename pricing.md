## 3. 課金モデル

### 3.1 決済手段
- クレジットカード（Stripe Checkout）
- Consultant年額のみ：請求書払い対応

### 3.2 課金サイクル
- 月初〜月末の暦月ベース
- 月途中のアップグレード：即時適用、日割り精算
- 月途中のダウングレード：月末で切り替え

### 3.3 利用制限の挙動

| 状況 | 挙動 |
|------|------|
| Free 月間診断5回超過 | 診断ボタン非活性 + アップグレード導線 |
| Free 11件以上入力 | バリデーション「Freeは10件まで」+ プラン比較リンク |
| Free でPDF出力タップ | モーダル「Proプランの機能です」+ プレビュー画像 |
| Free で4件目の履歴保存 | 最古の履歴を上書きする旨の通知 |
| Pro → Free ダウングレード | 月末までPro機能を利用可。翌月からFree制限 |
| 解約 | 月末まで利用可。翌月からFreeプランに自動移行 |

## 4. フリーミアム戦略

### 4.1 Proへの転換ポイント

| タイミング | 表示内容 |
|-----------|---------|
| 5回目の診断完了後 | 「今月の無料枠を使い切りました。Proなら無制限に診断できます」 |
| 11件以上入力時 | 「Freeは10件まで。Proなら50件まで一括診断」 |
| PDF出力ボタンタップ | 「PDFレポートはProプランの機能です」+ サンプルPDFプレビュー |
| 前回比較を見ようとした時 | 「改善率の確認はProプランで利用できます」 |
| 4件目の履歴 | 「Freeは直近3件まで。Proなら無制限に保存」 |

### 4.2 Consultantへの転換ポイント

| タイミング | 表示内容 |
|-----------|---------|
| Pro 4名目のチームメンバー追加時 | 「Proは3名まで。Consultantなら20名まで」 |
| 51件以上入力時 | 「Consultantなら100件まで一括診断」 |
| カスタムテンプレート登録時 | 「独自テンプレートはConsultantプランの機能です」 |

### 4.3 トライアル
- Proプランの14日間無料トライアル（クレジットカード登録不要）
- トライアル終了3日前・1日前・終了時にメール通知
- トライアル終了後は自動でFreeプランに移行（自動課金なし）
