## 4. ユーザー動線

### メインフロー（正常系）

```
1. トップページにアクセス
2. （任意）業種を選択
3. （任意）部署名を入力
4. 業務内容を入力（1行1業務）
5. 「診断する」ボタンを押す
6. ローディング画面を確認（数秒〜十数秒）
7. 診断結果を確認
8. 必要に応じてCSVをダウンロード
9. 「もう一度診断する」で再診断、またはブラウザを閉じて終了
```

### 異常系・分岐

| 状況 | 挙動 |
|------|------|
| 業務未入力で送信 | ボタン非活性。押せない |
| 入力が1行だけ | 正常動作。1件でも診断可 |
| 入力が50行超 | バリデーションエラー「50件以内で入力してください」 |
| API通信エラー | エラーメッセージ + リトライボタン表示 |
| AI応答が不正 | 「診断できませんでした」メッセージ + 再試行ボタン |
